# 电商用户行为数仓建设项目
基于 MySQL 8.0 + Python 实现的标准数仓项目，包含完整 ETL 流程、数据可视化、AI 智能分析与数据监控。

## 项目架构
- **ODS 层**：原始数据层，存储用户、订单原始数据
- **DWD 层**：明细数据层，清洗、过滤无效数据
- **DWS 层**：汇总数据层，按用户+日期聚合订单指标
- **ADS 层**：应用数据层，生成每日GMV、订单量、用户消费等业务报表

## 技术栈
- 数据库：MySQL 8.0
- 数据处理：Python、pymysql
- 数据可视化：Pyecharts
- AI 能力：火山引擎豆包大模型 API（自然语言转 SQL）
- 运维监控：Python 自动化监控脚本

## 核心功能
1.  **自动化造数**：生成 1000+ 用户、5000+ 订单测试数据，支持每日增量生成
2.  **数仓 ETL**：全流程自动化数据清洗、聚合、指标计算，一键刷新全链路
3.  **数据可视化**：生成交互式网页图表（GMV 趋势、订单量分析）
4.  **AI 智能分析**：自然语言转 SQL，自动分析业务数据，接入豆包大模型
5.  **数据监控**：全链路数据质量监控，异常自动预警，企业级运维标准

## 项目成果
构建完整电商数仓，可支撑日常业务分析、运营决策、数据可视化。
